# Engineering passes history

Historical status: prompt collection retired.

The former `ENGINEERING_PASSES.md` mixed one-off execution prompts, fixture-specific workflows,
shipping instructions, and performance hunts. Its reusable contributor rules now live in
`AGENTS.md`: preserve scope and dirty work, verify behavior in source/tests, keep framework concerns
outside the shared package, add focused behavioral tests, run the documented suites, and avoid
unrequested commits/releases. Git history preserves the original prompts if forensic detail is
needed.
